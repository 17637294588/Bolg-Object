二叉树数据结构

七月 10, 2016 发布在 后端, features

二叉树

完全二叉树：整棵树的高度和满二叉树高度一致，最下面的子节点从左向右依次排布，最下层不需要挂满子节点

满二叉树：整个树的每一层节点下面都挂满了子节点

二叉树的遍历

  • 深度优先遍历

深度优先搜索 (Depth First Search) 是沿着树的深度遍历树的节点，尽可能深的搜索树的分支

  • 先序遍历

    先访问根节点，然后递归使用先序遍历访问左子树，再递归使用先序遍历访问右子树 根节点->左子树->右子树

  • 中序遍历

    中序遍历访问左子树，然后访问根节点，最后再递归使用中序遍历访问右子树 左子树->根节点->右子树

  • 后续遍历

    后序遍历访问左子树和右子树，最后访问根节点 左子树->右子树->根节点

  • 广度优先遍历

从树的root（根节点）开始，从上到下从从左到右遍历整个树的节点

封装类模拟二叉树

'''
    1:定义一个节点类，有父节点，左节点，右节点
    2：定义一个构造树的类，初始化一个节点
    3：定义添加节点方法，接受一个节点变量
    4：先添加一个父节点，再添加左节点，右节点，当三个节点都存在时，再添加时就让将左右节点添加到 queue里
    5：广度优先遍历，定义两个:列表，一个用来与节点对象创建关系，一个用来存放节点
    6：深度优先遍历，利用递归实现的先序，中序，后续遍历
'''
class Node(object):
    def __init__(self,item):
        # 左节点
        self.lchild = None
        # 根节点
        self.item = item
        # 右节点
        self.rchild = None

# 构造二叉树
class Tree:

    """树类"""

    def __init__(self, root=None):

        self.root = root

    def add(self, item):

        node = Node(item)

        # 先添加一个根节点
        if not self.root:

            self.root = node
            return

        # 定义队列暂时存储节点
        queue = [self.root]
        a = 0
        while queue:
            a += 1
            print(a)
            cur = queue.pop(0)
            # print(queue)      # 此时列表是空
            if not cur.lchild:
                # 定义左节点
                cur.lchild = node
                return

            elif not cur.rchild:
                # 定义右节点
                cur.rchild = node
                return

            else:           
                queue.append(cur.rchild)
                # print(queue)
                queue.append(cur.lchild)
                print(queue)

if __name__ == "__main__":
    
    tree = Tree()
    for i in range(3):
        tree.add(i)

此时就构建成一个基本的拥有一个 父节点，一个左节点和一个右节点的二叉树

我们实现两种二叉树遍历 广度优先遍历和深度优先遍历

'''
    在上面二叉树类中添加广度优先方法
'''
#广度优先遍历 (层次遍历)
    def floor_travel(self):
        '''
            现将根节点存进列表里，再依次存子节点
        '''
        #如果是空树则直接返回一个[]
        if not self.root or self.root.item is None:
            return []
        else:
            my_queue = []
            #构造个容器来返回遍历的结果
            re_queue = []
            my_queue.append(self.root)
            while my_queue:
                cur_node = my_queue.pop(0)
                # print(my_queue)               # 此时 my_queue 弹出来后，列表里是空的
                re_queue.append(cur_node.item)       # 将根节点先存进去
                if cur_node.lchild:
                    my_queue.append(cur_node.lchild)
                if cur_node.rchild:
                    my_queue.append(cur_node.rchild)
            return re_queue


'''
    然后在二叉树中添加深度优先的三种遍历方法
'''
# 先序遍历
    def preorder(self,root):
        '''
            根节点->左子树->右子树
        '''
        re_queue = []
        if root == None:
            return
        else:
            def loop(root):
                if root == None:
                    return
                
                re_queue.append(root.item)
                loop(root.lchild)
                loop(root.rchild)
            loop(root)
            return re_queue

   # 中序遍历
    def inorder(self,root):
        '''
            左子树->根节点->右子树
        '''
        re_queue = []
        if root == None:
            return

        else:
            def loop(root):
                if root == None:
                    return
                
                loop(root.lchild)
                re_queue.append(root.item)
                loop(root.rchild)
            loop(root)
        return re_queue

    # 后序遍历
    def enorder(self,root):
        '''
            左子树->右子树->根节点
        '''
        re_queue = []
        if root == None:
            return
        else:
            
            def loop(root):

                if not root:
                    return
                loop(root.lchild)
                loop(root.rchild)
                re_queue.append(root.item)

            loop(root)
        return re_queue

将定义的这些遍历方法添加到构造二叉树类中就可以运行 main 程序查看效果

if __name__ == "__main__":
    
    tree = Tree()
    for i in range(5):
        tree.add(i)
    # data = tree.floor_travel()  # 广度遍历
    # print(data)
    # data = tree.preorder(tree.root)    # 先序遍历
    # data = tree.inorder(tree.root)    # 中序遍历
    data = tree.enorder(tree.root)       # 后序遍历
    print(data)
